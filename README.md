# CodeTrace

Algorithm visualization tool for understanding sorting algorithms through step-by-step execution and visual feedback.

## Current Features

### Algorithms
- Bubble Sort
- Quick Sort
- Merge Sort
- Insertion Sort
- Selection Sort

### Visualization
- Canvas-based rendering with color-coded states
- Step-by-step execution with playback controls
- Adjustable animation speed
- Array size control (5-100 elements)

### Controls
- Play/pause animation
- Step forward/backward through execution
- Reset to beginning
- Generate new random array
- View algorithm implementation code
- Adjust array size
- Adjust animation speed

### UI
- Algorithm information cards with complexity analysis
- Real-time execution messages
- Responsive layout

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- Canvas API for visualization

## Architecture

The codebase follows clean architecture principles with separation of concerns:

- Custom hooks for state management and logic
- Small, focused components
- Utility modules for shared functionality
- Type-safe implementation throughout
